Graphical Tool for customizable Heatmaps

Published by Journal of Computational Science Education

A heatmap can be used to visualize or represent raw data in an easily digestible format that can enable users to understand their data. During Junior year, I was granted the opportunity to work on a bioinformatics project with a Centre College computer science professor developing a customizable heat map and a program to run protein compounds through AutoCad Vina software and Tkinter, a module in the Python standard library which serves as a GUI toolkit.

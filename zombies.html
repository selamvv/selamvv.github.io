Description

A botnet is a group of internet devices that perform operations at the behest of their controller. This program retrieves, runs and reports python scripts at the request of the host.

The zombies retrieves one or more tasks to run at the request of the command and control. The command and control can ask a specific Zombie to run, stop running, or return the report of a particular script.

The python scripts contain simple tasks such as counting and use the subprocess library to save the python scripts as files on the zombie site.

a botnet is composed of a command and control (C and C) system that allows the controller to order the internet devices (zombies) to perform actions and report the results of those actions

  • Zombies retrieve python scripts and run them at the request of C and C, Stop running scripts at the request of C and C and Report the results (output) of the python scripts to the C and C (upon request)
  • Command and Control can ask a specific or all Zombies to run a python script, stop running a python specific script, ask a specific Zombie to return the result of a script, either the output of the script or a message that the program is still running.

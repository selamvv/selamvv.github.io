A communication platform for everyone that provides a place where people can reach a consensus.

Description

Product Description

A communication platform inspired by Discord which upon downloading allows users to create an account, create groups and channels with different privileges, send invites, messages block and kick out users.

Implementation Description

A proxy pattern through the server class ensures that clients can easily communicate with the “backend” of our system through RMI.

Through the login() methods, the Server ensures that only clients that provide a valid username and password can talk to the backend and receive a User object to be associated with.

Any client only has the permissions that their associated user does

StoreToXML() method within the Server saves all the important data within the server, such as the known clients and the database attribute it has, and/or reading this data back in on startup

For the purpose of notifications, the Observer pattern is implemented
